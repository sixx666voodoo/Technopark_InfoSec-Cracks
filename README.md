## Структура 

В каждой папке:  
  - Оригинальный exe файл
  - *Crack* версия, которая при любых введенных данных говорит, что эти данные **верны**  
  - Реализован keygen, который генерирует верный *code* под ваш *username*  
  - Описание алгоритма
